# my_web_project1
HealthTrack — веб‑приложение для отслеживания веса, питания и тренировок с персональной статистикой и графиками прогресса. Создано на Python с использованием Flask.
Автор: Кагиев Анзор
Функционал
Регистрация и авторизация — безопасные учётные записи с хешированием паролей.

Профиль пользователя — ввод и редактирование данных: вес, рост, целевой вес, расчёт ИМТ.

Дневник питания — добавление продуктов и калорий, история записей.

Трекер тренировок — фиксация упражнений, веса и количества повторений.

Графики прогресса — визуализация динамики веса и суточного потребления калорий.

Приватность данных — каждый пользователь видит только свои записи.

Технологии
Бэкенд: Python 3.8+, Flask.

ORM: SQLAlchemy.

Аутентификация: Flask‑Login.

Хеширование паролей: bcrypt.

База данных: SQLite (для разработки), PostgreSQL/MySQL (для продакшена).

Фронтенд: HTML5, CSS3, Bootstrap 5, Chart.js (графики).

Шаблонизация: Jinja2.

Модели базы данных
User — учётные записи (логин, email, хешированный пароль).

Profile — персональные данные (вес, рост, целевой вес).

FoodEntry — записи о питании (продукт, калории, дата).

Workout — записи о тренировках (упражнение, вес, повторения, дата).
